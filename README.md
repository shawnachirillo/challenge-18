 ## Challenge 18 Google Books MERN App
## Description
This full-stack MERN (MongoDB, Express.js, React, Node.js) application allows users to search for books using the Google Books API, save their favorite books, and manage their personal collection. It incorporates user authentication with JWT and provides a seamless user experience through a responsive frontend interface.
